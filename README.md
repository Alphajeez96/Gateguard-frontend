## Installation

* clone this repo
* cd into the project folder and run `php -S localhost:8000`
* visit localhost:8000 in your web browser

## If laravel is installed 

* clone this repo
* cd into the project folder and run `php artisan serve`
* visit localhost:8000 in your web browser
